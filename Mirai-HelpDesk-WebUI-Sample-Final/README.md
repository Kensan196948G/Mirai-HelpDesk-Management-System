# WebUI-Sample7 - モダンUIサンプル

## 概要
WebUI-Sample6を参照し、よりモダンで見やすいWebUIサンプルです。左側サイドメニューは白色背景で、充実したメニュー項目を提供します。

## 特徴

### デザイン
- **モダンなサイドバーレイアウト**: 左側に白背景の固定サイドバー
- **カテゴリ分けされたナビゲーション**: 機能別にセクション分けされたメニュー
- **クリーンなカラーパレット**: 青を基調としたプロフェッショナルなデザイン
- **統一されたコンポーネント**: カード、ボタン、バッジなどのUI要素が統一されたデザイン
- **レスポンシブデザイン**: モバイル、タブレット、デスクトップに対応

## 使い方

### ローカルで開く
```bash
# ブラウザで直接開く
open index.html

# またはローカルサーバーを起動
python3 -m http.server 8000
# http://localhost:8000 にアクセス
```

### カスタマイズ

#### カラースキームの変更
`css/style.css`の`:root`セクションでカラー変数を変更：

```css
:root {
    --primary-color: #3b82f6;      /* メインカラー */
    --success-color: #10b981;      /* 成功カラー */
    --danger-color: #ef4444;       /* 危険カラー */
    /* ... */
}
```

#### メニュー項目の追加
`index.html`のサイドバーセクションに追加：

```html
<div class="nav-section">
    <div class="nav-section-title">新しいセクション</div>
    <a href="#" class="nav-item" data-page="new-page">
        <span class="nav-icon">🆕</span>
        <span class="nav-label">新しいページ</span>
    </a>
</div>
```

## レスポンシブブレークポイント

- **タブレット**: 768px - 1024px（サイドバートグル）
