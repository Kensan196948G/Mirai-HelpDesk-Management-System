# AI分類機能の使い方（エンドユーザー向け）

チケット作成時のAI分類機能の使い方を、一般社員（Requester）向けに簡潔に解説します。

---

## 🤖 AI分類機能とは

### できること

チケットを作成する際に、**件名と詳細を入力するだけ**で、AIが以下を自動で提案してくれます：

- ✅ **カテゴリ**（例: Microsoft 365 > Exchange Online）
- ✅ **優先度**（P1〜P4）
- ✅ **影響度**（個人 / 部署 / 全社）
- ✅ **緊急度**（低 / 中 / 高 / 即時）

**メリット:**
- ⏱️ **時間短縮**: 分類に悩む時間が不要に（約3-5分削減）
- 🎯 **正確**: 90%以上の精度で適切な分類を提案
- 🚀 **迅速な対応**: 適切な担当者に自動で割り当てられる

---

## 📝 使い方（3ステップ）

### ステップ1: 件名と詳細を入力

チケット作成画面で、以下を入力してください：

**件名**（5文字以上）
```
例: Outlookで添付ファイルが送信できない
```

**詳細**（10文字以上）
```
例: 5MBのPDFファイルを送信しようとするとエラーになります。
昨日から発生しています。
```

---

### ステップ2: AI分類を実行

1. 件名と詳細を入力すると、「**AI分類を実行**」ボタンが表示されます
2. ボタンをクリックします
3. **約3-5秒**で結果が表示されます

**表示例:**
```
┌────────────────────────────────────┐
│ 🤖 AI分類提案                       │
├────────────────────────────────────┤
│ カテゴリ: Exchange Online          │
│ 信頼度: 92%（🟢 高信頼度）          │
│                                    │
│ 優先度: P3                         │
│ 信頼度: 85%（🟢 高信頼度）          │
│                                    │
│ 影響度: individual（個人）          │
│ 緊急度: medium（中）                │
└────────────────────────────────────┘
```

---

### ステップ3: 提案を採用または手動入力

**提案が正しい場合:**
- 「**この提案を採用**」ボタンをクリック
- フォームに自動入力されます

**提案が間違っている場合:**
- 「**手動で入力**」ボタンをクリック
- 正しい値を手動で選択してください

---

## 🎨 信頼度バッジの見方

AI の提案には、「信頼度」が表示されます。色で判断してください：

| バッジ | 信頼度 | 意味 | 推奨アクション |
|--------|--------|------|--------------|
| 🟢 **緑** | 90%以上 | 非常に高い信頼度 | **そのまま採用してOK** |
| 🟡 **黄** | 70-90% | 中程度の信頼度 | **内容を確認してから採用** |
| 🔴 **赤** | 70%未満 | 低い信頼度 | **手動で入力を推奨** |

## 💡 使用例

### 例1: Outlookのメール送信エラー

**入力:**
```
件名: Outlookで添付ファイルが送信できない
詳細: 5MBのPDFファイルを送信しようとするとエラーになります。
```

**AI提案:**
```
カテゴリ: Microsoft 365 > Exchange Online
信頼度: 92%（🟢 高信頼度）

優先度: P3
信頼度: 85%（🟢 高信頼度）
```

**アクション**: 「この提案を採用」をクリック

---

### 例2: PCの起動トラブル

**入力:**
```
件名: PCの電源が入らない
詳細: 電源ボタンを押しても反応がありません。充電ランプも点灯しません。
```

**AI提案:**
```
カテゴリ: PC・端末 > ハードウェア障害
信頼度: 93%（🟢 高信頼度）

優先度: P2
信頼度: 88%（🟢 高信頼度）
```

**アクション**: 「この提案を採用」をクリック

---

### 例3: 新しいソフトウェアの質問（低信頼度）

**入力:**
```
件名: 新しい会計ソフトの使い方を教えてください
詳細: 先月導入された会計ソフトの操作方法がわかりません。
```

**AI提案:**
```
カテゴリ: アプリケーション > その他
信頼度: 65%（🔴 低信頼度）

優先度: P4
信頼度: 72%（🟡 中信頼度）
```

**アクション**: 「手動で入力」をクリック → 正しいカテゴリを選択

## 🔒 個人情報の保護

### 自動マスキング

チケットにメールアドレスや電話番号が含まれている場合、以下のメッセージが表示されます：

```
ℹ️ 個人情報検出
メールアドレスや電話番号が検出されたため、AI分析時にマスキングされました。
```

**意味**: 個人情報は自動で伏せ字（`[EMAIL_MASKED]`等）に置き換えられ、外部のAIサービスには送信されません。

**例:**
```
入力:
担当者: yamada@company.com、電話: 03-1234-5678

AI送信時:
担当者: [EMAIL_MASKED]、電話: [PHONE_MASKED]
```

**安心してご利用ください**: 個人情報保護法に準拠した設計です。

## ❓ よくある質問

### Q: AI分類機能は必須ですか？

**A**: いいえ、オプションです。

従来通り、手動でカテゴリや優先度を選択することもできます。AI機能は、入力の手間を省くための補助ツールです。

---

### Q: AI提案が間違っていることはありますか？

**A**: はい、約10%の確率で間違うことがあります。

AI の精度は約90%です。間違っている場合は、「手動で入力」を選択し、正しい値を入力してください。

---

### Q: AI分類を使うと、対応が早くなりますか？

**A**: はい、早くなります。

AIが適切な担当者を自動で割り当てるため、迅速な対応が期待できます。

---

### Q: どのようなチケットでもAI分類できますか？

**A**: はい、すべてのチケットで使用できます。

ただし、新しい問題や特殊なケースの場合、信頼度が低くなることがあります。
