## 学習パス

### 初級者向け学習パス（1-2週間）

```
Week 1:
□ README.md を読む
□ QUICKSTART.md で実際に起動
□ 自分の役割のドキュメントを読む
□ 実際に触ってみる

Week 2:
□ 詳細ドキュメントを読む
□ 小さなタスクに挑戦
```

---

### 中級者向け学習パス（2-4週間）

```
Week 1-2: 初級者向けを完了

Week 3:
□ データベーススキーマを理解
□ API仕様書を読む
□ M365統合ガイドを読む

Week 4:
□ テストを書いてみる
□ 小さな機能を実装
```

---

### 上級者向け学習パス（継続的）

```
□ すべてのドキュメントを読む

□ セキュリティベストプラクティスを習得
□ 新機能の設計・実装

```

## よくある質問

### Q1: どのドキュメントから読めばいいですか？

**A**: まず [README.md](../../README.md) を読んでシステム全体を理解してください。その後、あなたの役割に応じたドキュメントに進んでください。

---

### Q2: ドキュメントが見つからない

**A**: このREADMEの「ドキュメントマップ」または「役割別ドキュメント」を参照してください。それでも見つからない場合は、プロジェクトルートの `README.md` をご確認ください。

---

### Q3: ドキュメントが古い・間違っている

**A**: GitHub Issuesで報告するか、プルリクエストで修正してください。ドキュメントの改善は大歓迎です。

---

### Q4: 開発環境のセットアップでつまずいた

**A**: 以下を順に確認してください:
1. [QUICKSTART.md](../../QUICKSTART.md)
2. [開発環境セットアップ.md](../03_開発者向け/開発環境セットアップ.md)
3. [SETUP.md](../SETUP.md)
4. [トラブルシューティング.md](../04_運用・保守/トラブルシューティング.md)
