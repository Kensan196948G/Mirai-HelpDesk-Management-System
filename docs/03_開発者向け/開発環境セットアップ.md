# 開発環境セットアップ

> **Note**: 詳細なセットアップガイドは [/docs/SETUP.md](../SETUP.md) および [/QUICKSTART.md](../../QUICKSTART.md) を参照してください。

## 前提条件

- Node.js 18+
- PostgreSQL 14+
- npm or yarn
- Git

## クイックスタート

### 1. リポジトリのクローン

```bash
git clone https://github.com/Kensan196948G/Mirai-HelpDesk-Management-System.git
cd Mirai-HelpDesk-Management-System
```

### 2. データベースセットアップ

```bash
# PostgreSQL データベース作成
createdb -U postgres mirai_helpdesk

# マイグレーション実行
psql -U postgres -d mirai_helpdesk -f database/migrate_all.sql
```

### 3. バックエンドセットアップ

```bash
cd backend
npm install

# 環境変数設定
cp .env.example .env
# .envを編集して適切な値を設定

# 開発サーバー起動
npm run dev
# http://localhost:3000
```

### 4. フロントエンドセットアップ

```bash
cd frontend
npm install

# 環境変数設定（必要に応じて）
# .env.localを作成

# 開発サーバー起動
npm run dev
# http://localhost:5173
```

## 開発ツール

### ESLint & Prettier

```bash
# ESLint実行
npm run lint

# Prettier実行
npm run format
```

### ホットリロード

開発サーバーはファイル変更を検知して自動的に再起動します。

- Backend: nodemon使用
- Frontend: Vite HMR使用

## トラブルシューティング

### ポート競合

```bash
# ポートが既に使用されている場合
# backend/.env
PORT=3001

# frontend/.env
VITE_PORT=5174
```

### データベース接続エラー

```bash
# PostgreSQLが起動しているか確認
sudo systemctl status postgresql

# 接続テスト
psql -U postgres -d mirai_helpdesk
```
