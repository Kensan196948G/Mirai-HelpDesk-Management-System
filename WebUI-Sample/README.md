# Mirai ヘルプデスク管理システム - WebUIサンプル

このフォルダには、ヘルプデスク運用システムのWebUIサンプルが含まれています。

## 概要

要件定義書に基づいて作成された、HTML + CSS + JavaScript による **シングルページアプリケーション（SPA）** のプロトタイプWebUIです。実際のバックエンドAPIとは接続されておらず、サンプルデータを使用した動作デモとなります。

## ファイル構成

```
WebUI-Sample/
├── css/
│   └── common.css          # 共通スタイルシート（アコーディオンメニュー対応）
├── js/
│   ├── common.js           # 共通JavaScript（サンプルデータ、ユーティリティ関数）
│   └── app.js              # メインアプリケーション（ページ切り替え、メニュー制御）
├── index.html              # メインHTML（SPA）
└── README.md               # このファイル
```

## 主な特徴

### ✨ シングルページアプリケーション（SPA）
- **1つのHTMLファイル**で全機能を提供
- JavaScriptによる動的なページ切り替え
- ページ遷移が高速でスムーズ

### 🎯 アコーディオン式サイドメニュー
- セクションをクリックで開閉可能
- 視認性が高く、スペースを有効活用
- スムーズなアニメーション

- **Microsoft 365**: Exchange (8), Teams (7), OneDrive (5), ライセンス (3), アカウント (4)
- **PC関連**: ハードウェア (8), ソフトウェア (5)
- **ネットワーク**: 接続・VPN関連 (5)
- **アプリケーション**: Adobe, Zoom, Chrome等 (5)
- **その他**: セットアップ、退職者対応等 (5)
- **高速検索**: キーワード、カテゴリで即座にフィルタリング
- **全項目に解決方法を記載**: 自己解決率の向上

### 👥 マルチロール対応
- **一般ユーザー**: ダッシュボード、チケット管理、ナレッジベース
- **エージェント**: チケット管理、受信トレイ、統計
- **M365オペレーター**: 作業タスク、実施履歴、SODチェック
- **管理者**: KPIダッシュボード、SLA管理、監査ログ

### 🎨 明るく見やすいデザイン
- グラデーション背景の美しいサイドバー
- 明るい配色でストレスフリー
- ホバーアニメーションで操作感向上

## 使い方

### 1. ブラウザで開く
```
C:\temp\Mirai-HelpDesk-Management-System\WebUI-Sample\index.html
```
をブラウザで開いてください。

### 2. ロール切り替え
ダッシュボード下部の「デモ: ユーザーロール切り替え」ボタンで、各ロールを体験できます：
- **一般ユーザー** → チケット作成、マイチケット表示
- **エージェント** → チケット管理、フィルタリング
- **M365オペレーター** → 作業タスク管理、実施ログ
- **管理者** → KPIダッシュボード、統計分析

### 3. アコーディオンメニュー
- サイドバーのセクションタイトルをクリックで開閉
- よく使う機能だけ展開して使用可能

## 利用可能なページ

### 一般ユーザー向け
- **ダッシュボード**: 統計情報、マイチケット一覧、人気のナレッジ
- **マイチケット**: 自分のチケット一覧（フィルタ・検索機能付き）

- **期限管理**: SLA期限の監視

### M365オペレーター向け
- **作業タスク**: M365操作タスク一覧

### 管理者向け
- **KPIダッシュボード**: SLA達成率、平均対応時間
- **分析レポート**: チケット傾向分析
- **SLA管理**: 優先度別の達成状況
- **監査ログ**: 操作履歴の確認
- **PCヘルプデスク一覧**: 全問い合わせ項目のマスタ

## 技術的な特徴

- **SPA（Single Page Application）**: ページリロードなしで画面遷移
- **クライアントサイドルーティング**: JavaScriptでページ管理
- **動的コンテンツレンダリング**: 各ページのHTMLを動的生成

### メニューシステム
- **役割ベースメニュー**: ユーザーロールごとに異なるメニュー構造
- **アコーディオン**: CSSトランジションによるスムーズな開閉
- **アクティブ状態管理**: 現在のページをハイライト

### スタイリング
- **CSSカスタムプロパティ**: 一貫した配色管理
- **グラデーション**: 視覚的に美しいUI
- **トランジション**: スムーズなアニメーション効果

## カスタマイズ

### カラーテーマの変更
`css/common.css` の `:root` セクションでCSS変数を変更できます。

### サンプルデータの変更
`js/common.js` の以下の変数を編集してください：
- `sampleTickets` - チケットデータ
- `sampleM365Tasks` - M365作業タスク
- `sampleKnowledge` - ナレッジ記事

### メニュー構造の変更
`js/app.js` の `menuStructure` オブジェクトでメニューをカスタマイズできます。

### 新しいページの追加
1. `js/app.js` の `pages` オブジェクトに新しいページを追加
2. レンダリング関数を作成
3. メニュー構造に追加

## 実装されていない機能

このサンプルは画面デザインとUIフローのプロトタイプです。以下の機能は実装されていません：

- バックエンドAPIとの通信
- 実際の認証・認可
- データベース連携
- ファイルアップロード（実際の保存）
- メール通知
- リアルタイム更新
- Microsoft Graph API連携
- 実際のSLA計算エンジン
- 監査ログの永続化
